# Gemini Project Review Prompt

Use this prompt when asking Gemini for a second opinion on product direction, user experience, usefulness, onboarding, and overall project clarity.

---

## Role

Gemini acts as an external second-opinion reviewer for:

- product usefulness
- user experience
- beginner friendliness
- clarity
- onboarding
- feature value
- prompt-system quality
- product potential

---

## Prompt

```txt
You are acting as an external product, UX, and AI workflow reviewer.

Project:
[project name]

Project purpose:
[paste short project description]

Current state:
[paste repo summary, README, roadmap, issue, or feature plan]

Target users:
[paste target users, or say "early AI builders / creative tech learners / prompt engineering users"]

Review this for:
- usefulness
- clarity
- user experience
- onboarding quality
- product potential
- feature prioritization
- whether the project feels understandable to a new user
- whether the AI workflow makes sense

Give me:

1. Overall assessment

2. What is most useful about this project

3. What is confusing or unclear

4. What would make this more valuable to users

5. What should be simplified

6. What should be improved next

7. Product ideas this could grow into

8. One concrete recommendation for the next step
```

---

## Best used for

- Second opinion on product direction
- UX and onboarding review
- Reviewing README/roadmap clarity
- Reviewing whether a repo feels useful to others
- Reviewing prompt-system/product potential
- Getting a different perspective before building
