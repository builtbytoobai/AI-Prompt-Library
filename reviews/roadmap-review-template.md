# Roadmap Review Template

Use this when deciding what to build next across one repo or across the full builtbytoobai project ecosystem.

---

## Template

```txt
Review this project roadmap as an external product and architecture reviewer.

Project or repo:
[project/repo name]

Current state:
[paste current README, roadmap, or summary]

Possible next features:
[paste feature ideas]

Long-term goal:
[paste long-term direction]

Review for:
- product value
- technical value
- learning value
- portfolio value
- difficulty
- risk
- dependencies
- whether the order makes sense

Give me:

1. Recommended next 3 priorities

2. What should be delayed

3. What should be removed or avoided

4. What should be split into smaller tasks

5. Best next GitHub Issue to create

6. Best next Claude Code task

7. What would make this more product-like

8. What would make this more useful for real users
```

---

## Best used for

- Choosing the next feature
- Planning repo direction
- Avoiding random feature creep
- Turning project ideas into a structured roadmap
- Moving from portfolio projects toward product ideas
