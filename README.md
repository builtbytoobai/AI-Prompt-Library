# AI Prompt Library

A curated collection of AI prompts for creativity, productivity, coding, and storytelling.

Built by curiosity. Powered by AI.

---

## 📌 About

This repository is a growing prompt library for exploring how AI can support creative work, technical learning, and idea development.

The goal is to collect useful prompts, workflows, and examples that can be reused, improved, and expanded over time.
