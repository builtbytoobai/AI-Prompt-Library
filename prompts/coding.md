# Coding Prompts

Prompts for learning programming and improving projects.

---

## Explain Code Prompt

```txt
Act as a patient programming teacher.

Explain this code in simple terms:

[insert code]

Break it down by:
- what the code does overall
- what each section does
- any important concepts I should understand
- one suggestion for improving it
```

---

## Improve Project Prompt

```txt
Act as a senior developer reviewing a beginner project.

Review this project idea or code:

[insert project/code]

Give me:
- what is good
- what could be improved
- one simple next feature
- one thing I should learn next
```
