# AI Project Planning Prompts

Prompts for turning ideas into small AI projects, experiments, and GitHub-ready builds.

---

## Idea to AI Project Prompt

```txt
Act as an AI product strategist and beginner-friendly technical mentor.

Help me turn this idea into a small AI project:

Idea: [insert idea]

Give me:
- a clear project name
- the smallest useful version
- the main feature
- 3 possible future features
- suggested tech stack
- folder structure
- README outline
```

---

## MVP Scope Prompt

```txt
Act as a practical product builder.

Help me reduce this project idea to the smallest version I can actually build:

Project idea: [insert idea]

Give me:
- what to include in v1
- what to avoid for now
- what makes the project useful
- what makes the project too complicated
- a realistic stopping point
```

---

## GitHub Repo Planning Prompt

```txt
Act as a senior developer helping a beginner structure a GitHub project.

Help me plan a clean GitHub repo for this project:

Project: [insert project]

Give me:
- repository name
- short description
- folder structure
- required files
- README sections
- good commit milestones
- possible topics/tags
```
