# Idea to Project Workflow

A simple workflow for turning an idea into a small buildable project.

---

## Step 1: Define the idea

```txt
What problem, feeling, or creative need does this idea respond to?
```

---

## Step 2: Define the smallest version

```txt
What is the simplest version I can build in one session?
```

---

## Step 3: Choose the format

Examples:

- Python script
- Prompt library
- Web page
- Automation
- Creative experiment

---

## Step 4: Build the first version

Focus on:

- making it work
- documenting what it does
- keeping it simple

---

## Step 5: Improve later

Do not overbuild version 1.

Ship first. Improve second.
